# File Organizer

A Python script that automatically organizes files in a directory by categorizing them into folders based on their file extensions.

## Features

- Organizes files into categories: HTML, Images, Videos, Documents, Archives, Audio
- Dry-run mode to preview changes before execution
- Handles file name conflicts automatically
- Detailed logging of all operations
- Summary statistics after organization

## Usage

1. Run the script:
```bash
python file_organizer.py
```

2. Enter the path of the folder you want to organize
3. Choose whether to do a dry run first
4. Review the summary

## Requirements

- Python 3.6+
- No external dependencies (uses standard library only)
